# PWA 图标资源

本目录用于存放 PWA 图标文件。

## 需要的图标

| 文件 | 尺寸 | 用途 |
|------|------|------|
| `icon-192.png` | 192×192 | Android 主屏幕图标 |
| `icon-512.png` | 512×512 | Android 主屏幕图标（高清） |

## 设计建议

- 使用品牌主色：`#4ECDC4`（青色）
- 可以包含发型相关的 emoji 或图标
- 背景建议使用渐变或纯色

## 临时解决方案

如果暂时没有设计资源，可以：
1. 使用在线的 favicon 生成器（如 favicon.io）
2. 从 App 中截取 🧚 表情符号作为图标
3. 使用纯色背景 + "发发" 文字

注意：图标缺失不影响 App 核心功能，只是无法"添加到主屏幕"。
